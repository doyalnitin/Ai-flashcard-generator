FlashMind

Turn any text or image into study flashcards — instantly.

AI-powered flashcard generator built with Express.js, Google Gemini, and deployed on Vercel.

02 / 12

The Problem

Before FlashMind

  • Manually writing each flashcard takes hours
  • Hard to know what's important to study
  • Cards are often too long or unfocused
  • No system for spaced repetition

With FlashMind

  • Paste text → flashcards in 5 seconds
  • AI extracts the key facts automatically
  • Each card = one atomic concept
  • Upload photos of notes or whiteboards

AI Prompt Engineering

System Instructions

You are an expert microlearning assistant.

Rules:
1. ONE concept per card
2. Use "QA" for questions
3. Use "CLOZE" for fill-in-the-blank
4. CLOZE: replace term with ______
5. Front must have clear context

JSON Schema Enforcement

{
  "cards": [{
    "type": "QA" | "CLOZE",
    "front": "string",
    "back": "string"
  }]
}

⚠️ Important Note

Vercel uses in-memory storage. Data resets on cold starts (~15 min idle). For persistent storage, connect a cloud database like Turso or Vercel Postgres.
